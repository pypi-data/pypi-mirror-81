Command: scipionwrite

Usage:
scipionwrite  model other-options

The scipionwrite command saves a map (volume data) or atomic coordinates to a file that will be automatically imported by Scipion. It also saves a ChimeraX session that may be recovered by the command scipionrs or the Scipion protocol chimera restore session.

Other-options

If the parameter prefix followed by a string is used then string is prepended to the name of the output file

Examples

scipionwrite #2 prefix dnabc

will save the model #2 in a file that starts with the string dnabc

scipionwrite #2.1 prefix sub_dna

will save the sub-model #2.1 in a file that starts with the string sub_dna
